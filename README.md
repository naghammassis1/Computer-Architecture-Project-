📦 Bin Packing Problem Solver (MIPS Assembly)

📌 Overview

This project implements a solution to the Bin Packing Problem using First Fit and Best Fit algorithms. The goal is to efficiently pack items into the minimum number of bins.

⚙️ Features

Solves bin packing using:

First Fit algorithm

Best Fit algorithm

Reads input data and processes item sizes

Compares packing efficiency between algorithms

🧪 Functionality

The program assigns items to bins based on the selected algorithm and outputs the bin allocation and usage.

🛠️ Technologies Used

MIPS Assembly

File Handling

Algorithms (First Fit, Best Fit)

▶️ How to Run

Open the program in a MIPS simulator (e.g., MARS or QtSPIM)

Load the assembly file

Run the program

Provide input file 

Algorithm comparison
